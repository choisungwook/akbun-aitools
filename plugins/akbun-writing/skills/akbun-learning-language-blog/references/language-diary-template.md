# 언어공부 일기 템플릿

생성할 Markdown 초안은 아래 구조를 사용한다.

```markdown
# {YYYY년 M월 D일 또는 주차} {영어공부|일본어공부} 기록

공부한 날짜: {YYYY년 M월 D일}

오늘은 {짧은 일기 문단}. 처음에는 {가벼운 감정이나 상황}이었는데, 정리하면서 {오늘의 핵심 표현}이 기억에 남았다.

## 오늘 배운 표현

- `{표현}`: {자연스러운 한국어 뜻 또는 사용 상황}
  - {원본 예문이 있으면 유지하거나 살짝 다듬은 예문}

## 헷갈렸던 표현

{헷갈린 표현이나 문장을 1-3문단으로 정리한다. 정답을 강의하듯 쓰지 말고, 내가 어떻게 이해했는지와 다시 볼 포인트를 적는다.}

## 다시 써보고 싶은 문장

- {복습용 문장 1}
- {복습용 문장 2}
- {복습용 문장 3}
```

`요약`, `목차`, `마무리`, `참고자료`라는 섹션은 추가하지 않는다.
마지막은 복습용 문장 목록으로 자연스럽게 끝낸다.
