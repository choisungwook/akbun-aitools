# 언어공부 일기 스타일

교재, 강의 노트, 기술블로그가 아니라 한국어 개인 공부 일기처럼 쓴다.

## 목소리

- `오늘은`, `저는`, `제가`, `다시 보니`, `기억에 남았다`처럼 존댓말이지만 개인적인 한국어를 쓴다.
- 첫 문단은 일기처럼 쓴다. 무엇을 공부했는지, 무엇이 낯설었는지, 정리하면서 무엇이 조금 분명해졌는지 적는다.
- 톤은 가볍고 구체적으로 유지한다. 감정을 과장하지 않는다.
- 홍보 문구나 게시/업로드를 전제로 한 문구는 피한다.

## 언어 표현 처리

- 명백한 오타나 깨진 제어문자가 아니라면 원본의 영어/일본어 표현은 그대로 보존한다.
- 원본의 어색한 한국어 설명은 자연스러운 한국어로 고친다.
- 긴 문법 강의를 새로 만들지 않는다. 일기로 읽히는 데 필요한 만큼만 설명한다.
- 일본어 원본에 한자/가나 읽기가 이미 있으면 보존한다. 별도의 전체 발음표를 새로 만들지 않는다.

## 개인정보와 기관명

- 공개 초안에는 회사명, 수업명, 학교명, 선생님 이름, 특정 기관명을 드러내지 않는다.
- 원본의 태그, 제목, 본문에 그런 이름이 있으면 `수업`, `공부 시간`, `학습 노트`처럼 일반화하거나 생략한다.

## 포맷

- H1 제목은 하나만 쓰고, 본문 섹션은 H2를 사용한다.
- 짧은 문단과 간결한 글머리표 목록을 우선한다.
- 사용자가 명시적으로 표를 요청하지 않으면 큰 표는 만들지 않는다.
- `요약`, `목차`, `마무리`, `참고자료`는 추가하지 않는다.
