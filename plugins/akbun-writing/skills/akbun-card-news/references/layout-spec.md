# 카드뉴스 레이아웃 스펙

## 기준 캔버스

모든 페이지는 세로형 4:5 비율을 사용한다. 기준 캔버스는 **1080 x 1350 px**이며, 아래 수치는 전부 이 캔버스 기준 px 값이다. 다른 해상도로 내보낼 때는 비율(%)을 유지한 채 스케일한다.

## 공통 스타일 토큰

| 토큰 | 값 | 용도 |
|---|---|---|
| `bg-beige` | `#E8E0D1` | 내용 페이지 배경 |
| `text-dark` | `#211D1A` | 내용 페이지 텍스트 |
| `text-light` | `#F4EFE6` | 표지 텍스트 (사진 위) |
| `photo-overlay` | `#000000`, opacity 0.25 | 표지 사진 위 가독성 오버레이 |
| `placeholder` | `#B0A897` | 사진 자리 표시 rect 색 |
| serif | Noto Serif KR (SIL OFL) | 헤드라인, 인용문 |
| sans | Pretendard (SIL OFL) | 질문 라벨, 캡션, 출처 |

## 페이지 1: 표지 (cover)

사진이 캔버스 전체를 채우고, 그 위 왼쪽 상단에 헤드라인과 서브 문단이 올라간다.

| 요소 | 위치·크기 | 스타일 |
|---|---|---|
| 배경 사진 | x 0, y 0, w 1080, h 1350 (full-bleed) | 위에 `photo-overlay` 덮기 |
| 헤드라인 | 좌 116, 상단 95부터 | serif 78px, line-height 1.15 (줄 간격 90), `text-light`, 최대 2~3줄 |
| 서브 문단 | 좌 116, 헤드라인 아래 약 60 간격 (2줄 헤드라인 기준 y 310부터) | serif 29px, line-height 1.45 (줄 간격 42), `text-light`, 2~4줄 |
| 우측 여백 | 116 | 텍스트 블록 최대 폭 848 |

## 페이지 2: 인용 카드 — 사진 포함 (quote-photo)

상단 오른쪽에 사진이 들어가고, 하단에 질문 라벨 + 큰 인용문 + 출처가 온다.

| 요소 | 위치·크기 | 스타일 |
|---|---|---|
| 배경 | 전체 | `bg-beige` |
| 사진 | x 498, y 63, w 522, h 414 (우측 여백 60) | 모서리 직각 |
| 질문 라벨 | x 81, y 630 부근 (첫 baseline 650) | sans 30px, `text-dark` |
| 인용문 | 텍스트 좌 85, 첫 baseline 760 | serif 58px, line-height 1.38 (줄 간격 80), `text-dark`, 최대 6줄 |
| 여는 따옴표 | x 56 (본문보다 왼쪽으로 매달림), 첫 줄과 같은 baseline | 인용문과 같은 serif 58px |
| 출처 이름 | x 81, baseline 1240 | sans 30px **bold**, `text-dark` |
| 출처 직함 | x 81, baseline 1278 | sans 30px regular, `text-dark` |

## 페이지 3: 인용 카드 — 텍스트 전용 (quote-only)

사진 없이 인용문만으로 채우는 페이지. 라벨이 페이지 맨 위로 올라가고 인용문 줄 수를 늘릴 수 있다.

| 요소 | 위치·크기 | 스타일 |
|---|---|---|
| 배경 | 전체 | `bg-beige` |
| 질문 라벨 | x 81, 첫 baseline 110 | sans 30px, `text-dark` |
| 인용문 | 텍스트 좌 85, 첫 baseline 240 | serif 58px, line-height 1.38 (줄 간격 80), `text-dark`, 최대 10줄 |
| 여는 따옴표 | x 56, 첫 줄과 같은 baseline | serif 58px |
| 출처 | 인용문 마지막 baseline + 90에 이름(bold), +38에 직함 | sans 30px, `text-dark` |

## 페이지 4: 사진 + 캡션 카드 (photo-caption)

상단에 큰 사진, 하단에 설명 문단이 오는 페이지.

| 요소 | 위치·크기 | 스타일 |
|---|---|---|
| 배경 | 전체 | `bg-beige` |
| 사진 | x 192, y 139, w 868, h 635 (좌 여백이 우 여백보다 넓어 오른쪽으로 치우침) | 모서리 직각 |
| 캡션 문단 | x 67, 첫 baseline 915 | sans 36px, line-height 1.5 (줄 간격 54), `text-dark`, 4~7줄 |

## 줄바꿈 규칙

- SVG `<text>`는 자동 줄바꿈이 없다. 줄마다 `<tspan>`을 나눠 넣는다.
- 인용문(58px)은 한 줄에 한글 기준 약 14~16자, 캡션(36px)은 약 24~26자를 넘기지 않는다.
- 단어(어절) 중간에서 줄을 끊지 않는다.
- 줄 수가 최대치를 넘으면 글을 줄이지 말고 페이지를 나눈다.
