# 카드뉴스 레이아웃 스펙

모든 페이지는 이 스펙을 그대로 따른다. 임의로 여백, 크기, 위치를 바꾸지 않는다.

## 캔버스

- 크기: **1080 × 1350** (세로형 4:5, 인스타그램 표준)
- 좌우 기본 여백: **76px** (폭의 약 7%)
- 모든 좌표는 이 캔버스 기준 절대값이다. 다른 크기가 필요하면 비율(% 값)로 환산한다.

## 표지 페이지 (cover)

사진이 전체를 채우고, 왼쪽 위에 브랜드 마크, 왼쪽 아래에 큰 제목, 아래 중앙에 핸들이 놓인다.

| 요소 | 위치·크기 | 비율 기준 |
|---|---|---|
| 배경 사진 | 0, 0, 1080 × 1350 전체 채움(full-bleed) | 100% |
| 제목 | x=76, 2줄, 크기 90px, 굵기 800, 흰색, 행간 122px | 크기 8.3% W, 행간 1.35 |
| 제목 1줄 baseline | y=915 | 67.8% H |
| 제목 2줄 baseline | y=1037 | 76.8% H |

- 제목은 **짧은 첫 줄 + 긴 둘째 줄** 2줄 구성이 기본이다. 1줄이면 baseline y=1037 하나만 쓴다.
- 사진이 밝아 흰 글자가 안 읽히면 하단 35% 영역에만 은은한 검정 그라데이션(불투명도 0 → 0.45)을 깐다. 기본은 그라데이션 없음.

## 내용 페이지 (content)

위쪽 절반은 사진, 아래쪽 절반은 검정 패널에 번호 제목과 본문이 놓인다.

| 요소 | 위치·크기 | 비율 기준 |
|---|---|---|
| 사진 | 0, 0, 1080 × 675 (상단 50%) | 50% H |
| 검정 패널 | 0, 675, 1080 × 675, 색 `#0D0D0D` | 하단 50% |
| 제목 | x=76, baseline y=810, 크기 50px, 굵기 700, 흰색 | 60% H, 크기 4.6% W |
| 본문 시작 | x=76, 첫 baseline y=890 | 66% H |
| 본문 글자 | 크기 30px, 행간 50px, 색 `#E8E8E8` | 크기 2.8% W, 행간 1.65 |
| 문단 간격 | 빈 줄 대신 다음 문단 첫 줄 baseline을 +80px (행간 50 + 30) | — |
| 본문 하단 한계 | 마지막 baseline y ≤ 1285 | 95% H |

- 제목은 `1. 제목`, `2. 제목`처럼 **번호 + 마침표 + 공백**으로 시작한다.
- 본문은 한 줄 40자 이내로 끊고, 문단은 2~4줄로 짧게 유지한다.
- **강조는 굵기 700 + 밑줄**이다. 밑줄은 baseline +14px 위치에 두께 2.5px 흰색 선으로 긋는다.
- 페이지당 문단 2~3개를 넘기지 않는다. 넘치면 페이지를 나눈다.

## 색

| 용도 | 값 |
|---|---|
| 검정 패널 | `#0D0D0D` |
| 제목·강조 텍스트 | `#FFFFFF` |
| 본문 텍스트 | `#E8E8E8` |
| 링크성 강조(선택) | `#7EC8E3` (밑줄 포함) |

## 폰트 (저작권 무료만 사용)

SVG의 `font-family`는 아래 순서로 지정한다. 전부 무료 라이선스(SIL OFL)라 상업적 사용이 가능하다.

- **Pretendard** — 기본. SIL OFL 1.1
- **Noto Sans KR** — 대체. SIL OFL 1.1

CSS로는 다음처럼 지정한다.

```css
font-family: Pretendard, 'Noto Sans KR', sans-serif;
```

유료 폰트, 라이선스 불명 폰트는 절대 지정하지 않는다. Figma/Canva에서 폰트가 없으면 사용자가 두 폰트 중 하나를 설치하거나 플랫폼 내 Noto Sans KR로 바꾸면 된다.
