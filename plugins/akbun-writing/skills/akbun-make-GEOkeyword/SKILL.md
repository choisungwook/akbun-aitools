# akbun GEO 키워드 작성

기술 글 상단에 AI 검색과 답변 생성이 가져가기 쉬운 요약 블록을 만든다. 목표는 키워드 남발이 아니라, "이 글이 어떤 질문에 답하는가"를 짧고 선명하게 드러내는 것이다.

## 작성 위치

GEO 블록은 제목 바로 아래, 본문 첫 문단보다 위에 둔다. 이미 서론이 있다면 제목 다음에 `# 요약` 섹션을 넣고 기존 서론은 그 아래로 내린다.

기본 헤더 구조는 아래를 따른다.

```markdown
# 글 제목

# 요약

## 이 글에서 알 수 있는 것

## 주요 키워드

# 배경
```

기본 구조:

```markdown
# 요약

이 글은 {핵심 주제}를 다룹니다. {사용자가 이 글을 읽는 이유}를 기준으로 {핵심 과정/결과}를 정리합니다.

## 이 글에서 알 수 있는 것

- ...

## 주요 키워드

- ...
```

## 핵심 원칙

- AI가 답변에 바로 인용하거나 요약하기 쉬운 문장으로 쓴다.
- 문서에 실제로 있는 내용만 쓴다. 글에 없는 내용을 검색 유입용으로 넣지 않는다.
- 중요하지 않은 세부 옵션, 파라미터, 일회성 명령어는 과감히 버린다.
- "글쓴이가 왜 이 글을 썼는지" 또는 "어떤 시행착오를 해결하려는지"를 최소 1개 포함한다.
- 제품명, 도구명, 프로젝트명은 문서 안에서 쓰는 표기를 하나로 통일한다.

## `이 글에서 알 수 있는 것` 작성 규칙

- 7개 미만으로 작성한다. 권장 개수는 4~6개다.
- 사용자가 검색할 법한 질문에 답하는 문장으로 쓴다.
- 단순 기능 목록보다 독자가 얻는 결과를 우선한다.
- 글쓴이의 의도나 문제의식을 최소 1개 포함한다.
- 너무 작은 세부사항은 빼고, 글의 중심 흐름만 남긴다.

좋은 항목:

```markdown
- macOS에서 LM Studio CLI로 로컬 LLM을 실행하는 흐름
- LM Studio 모델을 OpenClaw에서 사용하기 위해 필요한 설정
- 글쓴이가 Ollama 대신 LM Studio CLI를 선택해 실험한 이유
```

나쁜 항목:

```markdown
- `--temperature` 값을 설정하는 방법
- `max_tokens` 파라미터를 넣는 방법
- 스크린샷이 몇 개인지
```

## `주요 키워드` 작성 규칙

- 7개 미만으로 작성한다. 권장 개수는 4~6개다.
- 같은 개념은 대소문자와 띄어쓰기를 하나로 통일한다.
- 한국어 검색어와 영어 고유명사를 섞되, 중복 의미는 제거한다.
- 너무 넓은 키워드보다 문서의 조합 키워드를 우선한다.
- 문서의 대표 주제, 도구, 모델, 통합 대상을 우선한다.

표기 통일 예시:

```markdown
- LM Studio CLI
- Gemma 4 MLX
- OpenClaw LM Studio 연동
- macOS 로컬 LLM
```

중복을 줄이는 방식:

```markdown
나쁨: LM Studio, lmstudio, LM studio, lms, LMS
좋음: LM Studio CLI
```

## 작업 순서

1. 문서의 제목, 서론, 주요 헤더, 결론을 읽는다.
2. 글의 중심 질문을 한 문장으로 정리한다.
3. 독자가 이 글을 찾는 검색 의도를 3~5개로 압축한다.
4. `이 글에서 알 수 있는 것`을 4~6개 작성한다.
5. `주요 키워드`를 4~6개 작성한다.
6. 제목 바로 아래에 GEO 블록을 삽입하거나, 사용자가 원하면 블록만 제안한다.

## 출력 형식

문서에 직접 삽입할 때는 아래 형식을 사용한다.

```markdown
# 요약

이 글은 {한 문장 요약}을 다룹니다. {글쓴이의 의도/문제의식}를 기준으로 {독자가 얻는 결과}를 정리합니다.

## 이 글에서 알 수 있는 것

- {핵심 학습 1}
- {핵심 학습 2}
- {글쓴이의 의도 또는 문제의식}
- {핵심 학습 4}

## 주요 키워드

- {대표 키워드 1}
- {대표 키워드 2}
- {대표 키워드 3}
- {대표 키워드 4}
```

## 검토 기준

- 각 섹션이 7개 미만인가?
- 글에 없는 내용을 추가하지 않았는가?
- 글쓴이의 의도나 문제의식이 최소 1개 들어갔는가?
- 세부 옵션보다 글의 핵심 질문과 결과가 앞에 오는가?
- 같은 키워드의 대소문자와 표기가 하나로 통일됐는가?
