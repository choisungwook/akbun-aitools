# 카드뉴스 만들기

## 이 skill이 하는 일

사용자가 남긴 이미지와 글(또는 주제)을 카드뉴스로 만든다. 카드뉴스는 **표지 1장 + 내용 페이지 N장**으로 구성되며, 항상 세 가지를 출력한다.

1. **페이지별 카피** — 표지 제목, 내용 페이지의 제목·강조 소제목·체크 항목을 한국어로 정리한다.
2. **이미지 생성 프롬프트** — 각 페이지의 배경 사진을 GPT image, nano-banana 같은 이미지 생성 AI에 그대로 넣을 영어 프롬프트로 만든다.
3. **SVG 파일** — 페이지마다 1080x1350 SVG를 생성한다. 텍스트가 편집 가능한 상태로 유지되어 Figma/Canva에서 불러와 수정할 수 있다.

## 입력 처리

- **글을 준 경우**: 사용자의 글을 그대로 카피의 근거로 쓴다. 페이지 분량에 맞게 다듬되 없는 사실을 만들지 않는다.
- **글 없이 이미지·주제만 준 경우**: 이미지와 주제를 분석해 카피를 직접 생성한 뒤 이미지 프롬프트·SVG 작업까지 진행한다. 생성한 카피는 출력 맨 앞에 보여줘서 사용자가 고칠 수 있게 한다.
- **페이지 수**: 사용자가 지정하면 그대로 따른다. 지정이 없으면 내용 단위(장소, 항목, 단계)마다 내용 페이지 1장씩 + 표지 1장을 제안한다.

## 페이지 구성

### 표지 (1페이지)

- 배경 사진이 캔버스(1080x1350) 전체를 덮는다. 흰 여백이 보이면 안 된다.
- 사진 위에는 브랜드명·출처·페이지 번호 같은 글자를 얹지 않는다. 얹는 텍스트는 제목 하나뿐이다.
- 좌하단: 제목 최대 3줄, 흰색 굵은 글씨. 가독성을 위해 하단에 어두운 그라데이션 스크림을 깐다.
- 표지 제목은 `후킹 문구 + 숫자`가 잘 작동한다. 예: "주말이 달라진다! 올여름 부산에서 가볼 로컬 스팟 7"

### 내용 페이지 (2페이지~)

- 상단 2/3: 사진 영역. 사진 위에는 아무 글자도 얹지 않는다(출처·브랜드·페이지 번호 없음).
- 하단 1/3: 흰색 패널.
  - 큰 제목(한국어) + 오른쪽에 작은 영문 표기
  - 얇은 회색 구분선
  - 노란 형광펜 강조가 깔린 소제목 1줄
  - 체크박스 아이콘 + 본문 2~3줄
  - 마지막 줄은 위치핀 아이콘 + 주소/일정 같은 메타 정보 (없으면 생략)

정확한 좌표·크기·간격은 `references/layout-spec.md`를 따른다. **레이아웃 수치를 임의로 바꾸지 않는다.**

## 폰트 규칙

저작권 걱정 없는 무료 폰트만 쓴다.

- 기본: **Pretendard** (SIL OFL 1.1)
- 대체: **Noto Sans KR** (SIL OFL 1.1)

SVG의 모든 텍스트에 `font-family="Pretendard, 'Noto Sans KR', sans-serif"`를 지정한다. 출력 마지막에 "Figma/Canva에서 텍스트가 올바르게 보이려면 Pretendard 또는 Noto Sans KR 폰트가 설치(업로드)되어 있어야 한다"고 안내한다.

## SVG 작성 규칙

`assets/cover-template.svg`, `assets/content-template.svg`를 복사한 뒤 `{{...}}` 자리표시자를 채운다. 템플릿에 없는 요소를 임의로 추가하지 않는다.

- 캔버스: `width="1080" height="1350" viewBox="0 0 1080 1350"` 고정 (인스타그램 4:5).
- 텍스트는 `<text>` 요소로 남긴다. path로 아웃라인화하지 않는다. Figma/Canva가 편집 가능한 텍스트 레이어로 가져온다.
- 사진 영역은 기본적으로 회색 placeholder `<rect>` + 안내 문구로 둔다. 사용자가 실제 이미지 파일을 주고 삽입을 원하면 base64 `<image href="data:image/jpeg;base64,...">`로 교체한다.
- 사진 placeholder/이미지는 배경 영역과 크기가 정확히 같아야 한다. 표지는 x=0,y=0,w=1080,h=1350(캔버스 전체), 내용 페이지는 x=0,y=0,w=1080,h=900. 흰 여백이 남으면 안 된다.
- 모든 SVG의 첫 요소는 `<rect id="canvas" ... width="1080" height="1350" fill="#FFFFFF"/>`다. 사진·패널이 덮지 않는 빈 영역이 투명(렌더 시 배경색)으로 남지 않게 흰색 base를 깐다.
- 그룹(`<g>`)에 `id`를 붙여 Figma 레이어 이름이 되게 한다 (`background`, `title`, `panel` 등).
- 내용 페이지의 체크 항목이 2개면 한 줄을 지우고, 아래 행을 위로 당기지 말고 간격(68px)을 유지한 채 위에서부터 채운다.
- 노란 형광펜 rect의 `width`는 소제목 글자 수 × 폰트 크기(38px) × 0.95로 계산해 텍스트 길이에 맞춘다.
- 표지 제목이 1~2줄이면 남는 `<text>` 줄을 지우고, 남은 줄들을 아래 기준선(마지막 줄 y=1296)에 맞춰 내린다.

## 이미지 생성 프롬프트 규칙

페이지마다 배경 사진용 영어 프롬프트를 하나씩 만든다.

- 사실적인 사진 스타일을 기본으로 하되, 사용자가 준 참고 이미지가 있으면 그 무드(시간대, 색감, 구도)를 따른다.
- 세로 4:5 (1080x1350) 비율을 명시한다.
- **표지 프롬프트에는 반드시** "하단 1/3은 텍스트 오버레이를 위해 시각적으로 단순하게" 조건을 넣는다.
- 이미지 안에 글자·워터마크·로고가 생기지 않도록 금지 조건을 넣는다.
- 프롬프트마다 별도의 코드 펜스 블록(```text)에 담아 복사해 쓸 수 있게 한다.

프롬프트 뼈대는 다음을 따른다.

```text
A photorealistic vertical 4:5 photograph of <subject and scene>, <time of day and lighting>,
<mood and color palette>, <camera angle / composition>.
[Cover only] Keep the lower third of the frame visually simple and slightly dark so bold white
text can be overlaid on it.
DO NOT: no text, no letters, no watermarks, no logos, no people's identifiable faces in close-up.
```

## 작업 순서

1. **입력 파악.** 이미지·글·주제를 읽고, 글이 없으면 카피를 생성한다.
2. **페이지 설계.** 표지 1장 + 내용 페이지 N장을 정하고 페이지별 카피를 확정한다.
3. **이미지 프롬프트 작성.** 페이지마다 영어 프롬프트를 만든다.
4. **SVG 생성.** 템플릿을 복사해 자리표시자를 채우고 `cardnews/` 디렉터리(또는 사용자가 지정한 경로)에 `01-cover.svg`, `02-<슬러그>.svg` 순으로 저장한다.
5. **출력.** 페이지별 카피 → 이미지 프롬프트 블록들 → 생성한 SVG 파일 목록과 Figma/Canva 사용 안내 순으로 정리한다.

## 예시

입력 예시:

```text
부산 여행 카드뉴스 만들어줘. 흰여울문화마을이랑 광안리 해변 두 곳 소개할 거야.
```

출력 개요 예시 (카피 설계만 발췌):

```text
표지 — 제목: "바다가 부른다!" / "이번 주말 부산에서" / "가볼 로컬 스팟 2"
페이지 2 — 제목: 흰여울문화마을 (HUINNYEOUL VILLAGE)
  소제목: 절벽 위 골목마다 바다가 보이는 마을
  체크: 영도 해안 절벽을 따라 이어지는 컬러풀한 골목길 / 카페와 공방이 모여 있어 반나절 산책 코스로 좋음
  위치: 부산 영도구 영선동4가 (부산역에서 버스 20분)
페이지 3 — 제목: 광안리 해변 (GWANGALLI BEACH)
  ...
```

이후 페이지마다 이미지 프롬프트 블록과 SVG 파일(`01-cover.svg`, `02-huinnyeoul.svg`, `03-gwangalli.svg`)을 생성한다.

## 완료 전 확인

- 사용자가 준 글을 근거로 카피를 만들었는가? (글이 없을 때만 생성했는가?)
- 표지·내용 페이지가 `references/layout-spec.md`의 좌표·간격을 그대로 따르는가?
- 모든 텍스트에 Pretendard/Noto Sans KR font-family를 지정했는가?
- SVG 텍스트가 아웃라인화되지 않고 `<text>`로 남아 있는가?
- 사진 placeholder/이미지가 배경과 크기가 같아 흰 여백이 없는가? (표지 1080x1350, 내용 1080x900)
- 모든 SVG에 흰색 base rect(`id="canvas"`, 1080x1350)가 깔려 빈 영역이 투명하지 않은가?
- 사진 위에 브랜드명·출처·페이지 번호 같은 글자를 얹지 않았는가? (표지는 제목만, 내용 페이지는 사진 위 글자 없음)
- 표지 이미지 프롬프트에 "하단 1/3 단순하게" 조건이 있는가?
- 모든 이미지 프롬프트에 글자·워터마크 금지 조건이 있는가?
- 폰트 설치 안내를 출력에 포함했는가?
