# Markdown Writing Rules

## Header Rules

- Use H1 (`#`) only for the top-level title.
- All subsequent sections start from H2 (`##`).

## Code Block Rules

- Always add a description above a code block explaining what the code is.
- Never place a code block without a preceding description.
- Always specify the language type for every code block (e.g., ```bash,```json, ```yaml).
